# Library-Management-Mobile-Application
A simple Android-based Library Management Application developed in Java using Android Studio.
The application helps manage books in a library by allowing users to add, update, delete, and view books while also managing user accounts and settings.

*/ Features

* User Authentication -
Sign Up -
Login -
Forgot Password

* Book Management -
Add new books -
Update book details -
Delete books -
View book inventory 

* Account Management -
Change name -
Change email -
Manage user settings

* Additional Screens - 
Splash Screen -
Home Dashboard -
About Us page -
Feedback system -
Settings menu

* Technologies Used -
Java -
Android Studio -
Gradle Build System
XML Layouts
Android SDK
